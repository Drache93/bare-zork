# Bare Zork

Zork everywhere

## Build

```
npm run build:all
```

## Install

### MacOS

```
open ./build/darwin-arm64/bare-zork.app
```

### iOS

```
xcrun simctl install booted ./build/ios-arm64-simulator/bare-zork.app
```

### Adnroid

```
adb -e install build/android-arm64/BareZork.apk
```

Linux and Windows supported but not tested
